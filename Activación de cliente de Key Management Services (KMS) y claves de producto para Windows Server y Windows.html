Activación de cliente y claves de producto de Key Management Services (KMS)

  • Artículo
  • 04/19/2024

In this article

  1. Instalar una clave de producto
  2. Claves de licencia por volumen genéricas

Para utilizar KMS, necesita tener un host KMS disponible en su red local. Las computadoras que se activan con un host KMS deben tener una clave de producto específica. Esta clave a veces se denomina clave de cliente KMS, pero formalmente se la conoce como clave de licencia por volumen genérica de Microsoft (GVLK). Las computadoras que ejecutan ediciones de licencias por volumen de Windows Server y cliente de Windows son, de forma predeterminada, clientes KMS sin necesidad de configuración adicional, ya que el GVLK correspondiente ya está allí.

Sin embargo, hay algunos escenarios en los que necesitarás agregar el GVLK a la computadora que deseas activar en un host KMS, como por ejemplo:

  • Cómo convertir una computadora para que deje de usar una clave de activación múltiple (MAK)
  • Conversión de una licencia comercial de Windows a un cliente KMS
  • Si la computadora era anteriormente un host KMS

Importante

Para utilizar las claves enumeradas aquí (que son GVLK), primero debe tener un host KMS disponible en su red local. Si aún no tiene un host KMS, consulte cómo crear un host KMS para obtener más información.

Si desea activar Windows sin un host KMS disponible y fuera de un escenario de activación por volumen (por ejemplo, está intentando activar una versión comercial del cliente de Windows), estas claves no funcionarán . Deberá utilizar otro método para activar Windows, como utilizar una MAK o comprar una licencia minorista. Obtenga ayuda para encontrar su clave de producto de Windows y obtener información sobre las versiones originales de Windows .

Instalar una clave de producto

Si está convirtiendo una computadora desde un host KMS, MAK o una edición comercial de Windows a un cliente KMS, instale la clave de producto correspondiente (GVLK) de la lista de este artículo. Para instalar una clave de producto de cliente, abra un símbolo del sistema administrativo en el cliente, ejecute el siguiente comando y luego presione Enter:

Símbolo del sistema de Windows 
slmgr /ipk <product key>

Por ejemplo, para instalar la clave de producto para Windows Server 2022 Datacenter Edition, ejecute el siguiente comando y luego presione Enter:

Símbolo del sistema de Windows 
slmgr /ipk WX4NM-KYWYW-QJJR4-XV3QB-6VM33
